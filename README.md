# Bulk-PDF-to-PNG
A lightweight, browser-based tool that converts up to 20 PDF files to PNG images in one go no server, no uploads, no installs. Everything runs locally in your browser using PDF.js, so your files never leave your machine.

## Features
- Drag & drop or browse up to 20 PDFs at once
- Preserves original filenames (e.g. report.pdf → report.png)
- Multi-page PDFs export as filename_page1.png, filename_page2.png, etc.
- Three quality settings: 72 DPI (screen), 150 DPI (standard), 300 DPI (print)
- Real-time per-file progress tracking
- Zero dependencies to install — just open the HTML file in any modern browser
